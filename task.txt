Создать приложение с использованием одного из фреймворков Angular / React / Vue для просмотра текущей погоды в Челябинске, а также прогноза на ближайшие 5 дней, в качестве источника данных можно использовать (https://openweathermap.org). 
Требования: 
1)	Приложение должно один раз запрашивать у пользователя API-ключ погоды и сохранять для дальнейшей работы;
2)	Должен быть реализован постраничный вывод информации о прогнозе по 3 часа. На каждую страницу отдельный запрос. Компонент пагинации для вывода информации должен быть реализован без использования сторонних библиотек;
3)	Адаптивная верстка под разные расширения экрана.
4)	Визуальное оформление на ваше усмотрение без использования сторонних библиотек;
5)	Загрузить проект на github.com, желательно разместить приложение на github pages.
6)	*По желанию добавить возможность выбора места на карте для получения погоды.
